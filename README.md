# tree-sitter-mermaid

[Mermaid.js](https://mermaid-js.github.io/mermaid/#/) grammar for [tree-sitter](https://tree-sitter.github.io/tree-sitter/).

## Supported Syntax

There may be unsupported grammars for some types, even if they are marked as completed.

- [x] Class
- [ ] Er
- [ ] Flowchart
- [ ] Gantt
- [ ] Git
- [ ] Info
- [ ] Pie
- [ ] Requirement
- [x] Sequence
- [ ] State
- [ ] User-Journey
